# lizWeb
Simple presentation page for the architect Liz Alavarenga.

## Development
1. Vue.js
2. Javascript
3. vueuse/motion
4. Tailwind 
5. Swiper

## Installation
Project Cloning
```
git clone https://github.com/Yosiak-alv/lizWeb.git
cd lizWeb
```
Installing Dependencies:
```
npm install
```
run it:
```
npm run dev
```
